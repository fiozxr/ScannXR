Scannxr — Advanced Python Network Scanner

Powerful, modular and high-performance local network discovery & port-scanning toolkit. ARP, SYN stealth scans, TCP connect, banner grabbing, OUI vendor lookup, HTML/JSON reports and an optional Flask dashboard.

Quick start

Copy these commands to run Scannxr locally. See full docs below.

Get Started Usage Python 3.8+ Optional: scapy • flask

Basic scan (no root)

python3 scannxr.py -n 192.168.1.0/24

Save JSON & HTML

python3 scannxr.py -n 192.168.1.0/24 --json results.json --html report.html
Host discovery
ARP (fast), ICMP/TCP fallback, threaded sweep
Port scanning
SYN stealth (scapy, root) or TCP connect (fallback)
Fingerprinting
OS guess via TTL, banner grabbing, OUI vendor lookup
Reporting
JSON & HTML export + optional Flask UI

Install

git clone https://github.com/yourusername/scannxr.git

cd scannxr pip install -r requirements.txt

CLI options

OptionMeaning
-nNetwork CIDR (e.g. 192.168.1.0/24)
-pPorts (22,80,443 or 1-65535)
--modeauto / syn / connect
--jsonSave JSON
--htmlSave HTML report

Legal

Run only on networks you own or have permission to test. Unauthorized scanning may be illegal.

Installation

Requirements: Python 3.8+. Optional packages for full features:

pip install scapy flask

Usage examples

Basic

python3 scannxr.py -n 192.168.1.0/24

Custom ports

python3 scannxr.py -n 192.168.1.0/24 -p 22,80,443,8000-8100

Full port range

python3 scannxr.py -n 192.168.1.0/24 -p 1-65535

Stealth SYN scan (root)

sudo python3 scannxr.py -n 192.168.1.0/24 --mode syn

Save reports

python3 scannxr.py -n 192.168.1.0/24 --json out.json --html report.html

Output format

JSON example shows host metadata and open ports.

{

"ip": "192.168.1.10", "mac": "AA:BB:CC:11:22:33", "vendor": "TP-Link", "os": "Linux/Unix", "open_ports": {"22":"OpenSSH","80":"Apache"} }
