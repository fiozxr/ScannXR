Scannxr

Advanced Python-based network scanner with host discovery, stealth scanning, fingerprinting, reporting, and more.

Install Usage

Features

ARP Discovery
Ultra-fast LAN host detection.
SYN Stealth Scan
Root-only, low-noise port scan.
TCP Connect Scan
Works without root privileges.
Banner Grabbing
Identify services running on ports.
OS Guessing
Based on TTL and fingerprinting.
HTML/JSON Reports
Export professional scan results.
Flask Web UI
Optional live dashboard.

Installation

GitHub 

git clone https://github.com/fiozxr/scannxr
cd scannxr
pip install -r requirements.txt
Optional: 
pip install scapy flask

Usage Examples

Basic scan: 
python3 scannxr.py -n 192.168.1.0/24
Custom ports: 
python3 scannxr.py -n 192.168.1.0/24 -p 22,80,443,8000-8080
Stealth SYN scan (root): 
sudo python3 scannxr.py --mode syn -n 192.168.1.0/24
Export reports: 
python3 scannxr.py -n 192.168.1.0/24 --json out.json --html report.html
